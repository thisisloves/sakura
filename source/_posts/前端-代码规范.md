### 项目文件夹

1. 分层明白
2. 文件夹首字母大写小写统一

### JS 规范

1. 统一缩进
3. 字符串统一单引号
4. 变量和函数为小驼峰法标识, 即除第一个单词之外，其他单词首字母大写（ lowerCamelCase）
5. 全局变量为大写 (UPPERCASE )
6. 常量 (如 PI) 为大写 (UPPERCASE )

```js
let a = '23';
```

7. 每个功能函数 统一一注释
8. 函数定义结束不加分号，其他都加分号

### CSS 规范

1. 统一缩进
2. 属性名冒号后空格

```css
margin: 0;
```


3. 使用 - 来连接属性名

```css
.main-title {

}
```

4. 选择器多个选择器同时声明同一个样式时，每个选择器各占一行
5. 颜色统一 16 进制

### HTML 规范

1. 统一缩进
2. 引入 css 必须要有 rel，引入 javascript 无需声明 type，需放在页面末尾
3. img 必须增加 alt 属性
