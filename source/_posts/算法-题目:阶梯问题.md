### 阶梯问题

- 楼梯一共有 n 层，但你每次爬楼梯只能走一步或两步，计算共有多少种走法。

### 代码实现

1. n=0 和 n=1 的时候 并没有其他可选择的，所以可以得出 f(0)=0;f(1)=1。
2. n>=2 时情况就变复杂起来，但是这个时候可以操作的步骤也就 2 种也就是走 1 步(n-1)与走 2 步(n-2)。所以可以得到 f(n)=f(n-1)+f(n-2)。

```js
function getNum(m) {
  if (m === 1 || m === 2) {
    return m;
  } else {
    return getNum(m - 2) + getNum(m - 1);
  }
}
```
