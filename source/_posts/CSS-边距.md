### 边距类型

1. 外边距(margin)

- margin属性定义元素周围的空间。

```css
div{
   margin:auto;
   margin:1%;
   margin:25px 50px 75px 100px;
}
```


2. 内边距(padding)

- 定义元素边框与元素内容之间的空间，即上下左右的内边距。


```css
div{
   padding:1%;
   padding:25px 50px 75px 100px;
}
```
