### HTML 属性

1. HTML 元素可以设置属性
2. 属性可以在元素中添加附加信息
3. 属性一般描述于开始标签
4. 属性总是以名称/值对的形式出现，比如：name="value"。

### HTML 元素的种类

| 属性  |                              描述                               |
| :---: | :-------------------------------------------------------------: |
| class | 为 html 元素定义一个或多个类名（classname）(类名从样式文件引入) |
|  id   |                        定义元素的唯一 id                        |
| style |               规定元素的行内样式（inline style）                |
| title |              描述了元素的额外信息 (作为工具条使用)              |
| lang  |                    设置元素中内容的语言代码                     |
